GITHUB DEPLOYMENT INSTRUCTIONS
==============================

1. CREATE A NEW GITHUB REPOSITORY:
   - Go to https://github.com/new
   - Name your repository (e.g., "portfolio-website")
   - Make it public or private as desired
   - DO NOT initialize with a README
   - Click "Create repository"

2. COPY THE REPOSITORY URL:
   - It will look like: https://github.com/YOUR_USERNAME/portfolio-website.git

3. RUN THESE COMMANDS IN YOUR TERMINAL:
   cd c:\Users\tusha\Desktop\Webite
   git remote remove origin
   git remote add origin YOUR_REPOSITORY_URL_HERE
   git push -u origin main

4. VERIFY DEPLOYMENT:
   - Visit your GitHub repository URL
   - You should see all your files uploaded

Note: Replace YOUR_REPOSITORY_URL_HERE with the actual URL from step 2.

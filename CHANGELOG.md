# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [2.0.0] - 2025-12-03

### Added
- Upgraded to Django 5.2
- Updated all dependencies to latest versions
- Enhanced deployment configuration for better compatibility

### Changed
- Updated Python runtime to 3.12.9
- Updated requirements.txt with latest package versions
- Updated documentation to reflect new versions
- Improved build process in build.sh

### Fixed
- Resolved compatibility issues with newer Django versions
- Updated settings files to use Django 5.2 documentation references

## [1.1.0] - 2025-12-03

### Added
- Python 3.12.9 support
- Latest package versions in requirements.txt
- Enhanced deployment configuration

### Changed
- Updated from Django 3.1.12 to Django 4.2
- Updated all dependencies to their latest compatible versions
- Improved README documentation
- Enhanced build.sh script with pip upgrade step

### Fixed
- Resolved "ModuleNotFoundError: No module named 'cgi'" error
- Fixed Python version compatibility issues
- Corrected deployment configuration for Render

## [1.0.0] - 2025-12-01

### Added
- Initial release of the portfolio website
- Django 3.1.12 framework
- Python 3.9.16 runtime
- Full portfolio functionality with multiple sections
- Contact form with email notifications
- AI chat integration
- Service booking system
- Project showcase
- Responsive design with Bootstrap 5
- Deployment configurations for Render and Heroku
